This is a personal challenge of mine to code minesweeper in as many programming languages as possible. I generally start with a 5x5 grid, but I plan to add a way for users to adjust the size of the board.
Some requirements for what I consider playable minesweeper:
-Random board generation (not the same board every time).
-First tile must be guaranteed to not be a mine (mine generation occurs after first move).
-Flagging
